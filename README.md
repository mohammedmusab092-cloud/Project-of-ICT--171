# Project Overview

ZestraFit is a cloud-hosted fitness tracking platform developed and deployed as part of the ICT171 Cloud Server Project. The application is hosted on a Microsoft Azure Virtual Machine running Ubuntu Linux and configured using Nginx as the web server.

The project demonstrates practical skills in cloud deployment, Linux administration, web server configuration, database integration, DNS management, SSL/TLS security, and GitHub version control.

Users can create accounts, log in securely, maintain fitness records, submit support requests, and access a server status dashboard.

# DNS Configuration

A custom domain (zestra.fun) was registered and configured using DNS A records that point to the Azure Virtual Machine public IP address.

This allows users to access the application through a user-friendly domain name instead of directly using the server IP address.
