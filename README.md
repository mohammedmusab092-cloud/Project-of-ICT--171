# ZestraFit

Student Name: Muhammad Musab Qasir  
Student Number: 35991305  

ZestraFit is a cloud-hosted fitness tracking website deployed on an Azure Linux VM using Nginx, PHP, SQLite, DNS, and SSL/TLS.

## Features

- Fitness homepage
- User registration
- User login
- Fitness dashboard
- Workout, water, BMI, and step logs
- Support message page
- Server status dashboard

## Technologies

- Azure Linux VM
- Ubuntu
- Nginx
- PHP
- SQLite
- HTML
- CSS
- JavaScript
