## Project Overview

ZestraFit is a cloud-hosted fitness tracking platform developed for the ICT171 Cloud Project. The application allows users to register, log in, maintain fitness records, submit support requests, and access a server status dashboard.

## Infrastructure

* Cloud Provider: Microsoft Azure
* Operating System: Ubuntu Linux
* Web Server: Nginx
* Backend: PHP
* Database: SQLite
* Version Control: GitHub

## Website Features

* User Registration
* User Login & Logout
* Fitness Dashboard
* Workout Logs
* Water Intake Tracking
* BMI Records
* Step Tracking
* Support Contact System
* Server Status Dashboard

## DNS & Security

A custom domain (**zestra.fun**) was configured and connected to the Azure Virtual Machine using DNS records. HTTPS was enabled using SSL/TLS certificates to provide secure communication between users and the server.
